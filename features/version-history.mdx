    ```
    MAJOR.MINOR.PATCH
    ```

## API Integration

### Fetching Version Data

```typescript
import { getVersionHistory } from '@npmstats/api';

// Get version history
const history = await getVersionHistory({
  package: 'react',
  range: 'last-year',
  includePrerelease: false
});
```

### Comparing Versions

```typescript
import { compareVersions } from '@npmstats/api';

// Compare two versions
const comparison = await compareVersions({
  package: 'react',
  version1: '17.0.0',
  version2: '18.0.0',
  includeBreakingChanges: true
});
```

### Version Control

2. Maintain detailed changelogs
3. Document breaking changes
4. Plan deprecations
