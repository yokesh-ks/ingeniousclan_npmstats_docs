## Overview

The Similar Packages feature helps you discover and evaluate alternative NPM packages that serve similar purposes. Make informed decisions by comparing packages based on various metrics and characteristics.

## API Integration

### Finding Similar Packages

```typescript
import { findSimilarPackages } from '@npmstats/api';

// Get similar packages
const alternatives = await findSimilarPackages({
  package: 'moment',
  limit: 5,
  metrics: ['downloads', 'size', 'maintenance']
});
```

### Comparing Packages

```typescript
import { comparePackages } from '@npmstats/api';

// Compare multiple packages
const comparison = await comparePackages({
  packages: ['date-fns', 'moment', 'dayjs'],
  metrics: ['size', 'performance', 'features']
});
```
