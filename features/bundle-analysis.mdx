## Overview

Bundle analysis helps you understand the size impact of NPM packages on your application, enabling you to make informed decisions about dependencies and optimize your application's performance.

## API Integration

### Getting Bundle Analysis

```typescript
import { analyzeBundleSize } from '@npmstats/api';

// Analyze package bundle
const analysis = await analyzeBundleSize({
  package: 'react',
  version: '18.2.0',
  format: 'esm',
  includeSubmodules: true
});
```

### Comparing Versions

```typescript
import { compareBundleSizes } from '@npmstats/api';

// Compare package versions
const comparison = await compareBundleSizes({
  package: 'react',
  versions: ['17.0.0', '18.0.0', '18.2.0']
});
```
