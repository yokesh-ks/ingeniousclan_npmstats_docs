## Overview

Security Scanning helps you identify, assess, and remediate security vulnerabilities in your NPM packages and their dependencies. Stay ahead of security threats with real-time monitoring and automated alerts.

## API Integration

### Running Security Checks

```typescript
import { scanPackageSecurity } from '@npmstats/api';

// Perform security scan
const results = await scanPackageSecurity({
  package: 'your-package',
  version: 'latest',
  includeIndirect: true,
  severityThreshold: 'medium'
});
```

### Setting Up Alerts

```typescript
import { setupSecurityAlerts } from '@npmstats/api';

// Configure security monitoring
const alerts = await setupSecurityAlerts({
  package: 'your-package',
  notifications: {
    email: true,
    slack: true,
    webhook: 'https://your-webhook.com'
  },
  severity: ['critical', 'high']
});
```
