## Overview

Dependency Insights provides a comprehensive view of package dependencies, helping you understand relationships between packages, identify potential issues, and maintain a healthy dependency tree.

## API Integration

### Fetching Dependency Data

```typescript
import { analyzeDependencies } from '@npmstats/api';

// Get dependency analysis
const deps = await analyzeDependencies({
  package: 'your-package',
  version: 'latest',
  depth: 'full'
});
```

### Monitoring Dependencies

```typescript
import { watchDependencies } from '@npmstats/api';

// Set up dependency monitoring
const watcher = watchDependencies({
  package: 'your-package',
  onUpdate: (changes) => {
    console.log('Dependency updates:', changes);
  }
});
```
